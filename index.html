Czym jest analiza heurystyczna?

6. Rozpoznawanie zamiast przypominania

EN: "Minimize the user's memory load by making elements, actions, and options visible. The user should not have to remember information from one part of the interface to another. Information required to use the design (e.g. field labels or menu items) should be visible or easily retrievable when needed."

PL: Minimalizuj obciążenie pamięci użytkownika, sprawiając, by elementy, akcje i opcje były widoczne. Użytkownik nie powinien musieć zapamiętywać informacji podczas przechodzenia z jednej części interfejsu do drugiej. Informacje potrzebne do korzystania z systemu (np. etykiety pól lub elementy menu) powinny być widoczne lub łatwo dostępne w razie potrzeby.
